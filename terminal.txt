untuk seeder baru
---
php artisan db:seed

cara bikin database migration/model/controller
---
> php artisan make:migration nama_migrateion
> php artisan make:model NamaModel
> php artisan make:controller NamaController

untuk migrate
---
> php artisan migrate

untuk rollback / hapus migration terakhir
---
> php artisan migrate:rollback

daftarkan Helpers
---
Muat Ulang Autoloader Composer: Jalankan perintah berikut di terminal atau Command Prompt untuk memperbarui konfigurasi agar Laravel mengenali helper tersebut:
> composer dump-autoload
